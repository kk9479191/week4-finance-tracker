# Personal Finance Tracker

## Project Description

Personal Finance Tracker is a Python-based application that helps users manage and track their daily expenses. The application allows users to add expenses, store them permanently in a JSON file, search expenses by category, and generate reports for better financial management.

## Features

* Add new expenses
* Store expenses in JSON format
* Load saved expenses automatically
* Search expenses by category
* Generate monthly expense reports
* View category-wise expense breakdown
* User-friendly menu system
* Data persistence using file handling

3. Run the application

python run.py
